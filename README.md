# 🌿 EcoMonitor System

A full-stack ecological monitoring system designed for the multitemporal analysis of Andean ecosystems using Sentinel-2 data. 

This system features a robust REST API (FastAPI), a containerized relational database (MySQL), and a lightweight, dependency-free administrative frontend.

## 🚀 Prerequisites

To run this project on **any operating system** (Windows, macOS, or Linux), ensure you have the following installed:

* **[Docker Desktop](https://docs.docker.com/get-docker/)** (or Docker Engine + Docker Compose)
* **[Miniconda](https://docs.anaconda.com/free/miniconda/index.html)** (or standard Python 3.10+)
* **[Git](https://git-scm.com/downloads)**
